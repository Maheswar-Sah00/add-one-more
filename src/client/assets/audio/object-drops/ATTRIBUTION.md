# Object drop sounds — attribution & licence

All 15 object-drop sounds in this folder are **procedurally synthesised from
scratch** for "One More Thing". They are **original works**, not derived from any
third-party sample, recording, or commercial game asset.

- **Source:** generated by [`scripts/generate-drop-sounds.py`](../../../../../scripts/generate-drop-sounds.py)
  (Python + numpy + scipy) — additive tone bodies, exponential impact envelopes,
  and filtered-noise texture layers, mixed per object to match each material.
- **Licence:** **CC0 1.0 / public domain.** No attribution required; free to use,
  modify, and redistribute.
- **Modifications:** N/A (generated, not sourced). Re-generate at any time by
  running the script; output is deterministic (fixed RNG seed).
- **External dependencies at runtime:** none. Files are bundled locally via Vite
  and served from the app's own origin.

**Format:** mono, 22.05 kHz, 16-bit PCM WAV (no MP3/OGG encoder available in this
environment; files are small — ~8–26 KB each, ~218 KB total — and optimised for
fast web/mobile loading). Soft-limited to prevent clipping; normalised for
consistent perceived loudness, then per-object gain applied at playback.
